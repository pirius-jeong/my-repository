# 부록 D. Building an Agent with AgentSpace(AgentSpace 에이전트 구축)

## D.1 AgentSpace 개요

- **플랫폼**: **엔터프라이즈 에이전트 통합**(Gemini 기반).
- **핵심**: 디지털 자산 검색(문서·이메일·DB) + 지식 그래프.
- **기능**: 노코드 Agent Designer, A2A 프로토콜(다중 에이전트).
- **목표**: 자율 추론·계획·실행(보고서·오디오 요약).

---

## D.2 구축 단계 (UI)

1. **접속**: Google Cloud Console → AI 앱 → AgentSpace.
2. **연결**: Gmail·Calendar·Jira·Workday 등.
3. **프롬프트**: 갤러리 선택 or 사용자 정의.
4. **고급**: 데이터 저장소·Knowledge Graph·웹 노출·분석.
5. **채팅**: 완성 에이전트 인터페이스 테스트.

**스크린샷 흐름** (UI 중심): Console → 서비스 연결 → 프롬프트 편집 → 고급 설정 → 채팅.

---

## D.3 보안·확장

- **보안**: 역할 기반·암호화.
- **확장**: A2A로 복잡 워크플로.
